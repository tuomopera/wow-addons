# wow-addons

Personal World of Warcraft addons.

## GankList

Remember who ganked you while leveling, synced with a friend or two.

- **[Download GankList.zip](https://github.com/tuomopera/wow-addons/releases/latest/download/GankList.zip)** — extract the `GankList` folder into `World of Warcraft/_anniversary_/Interface/AddOns/`, restart WoW, enable it.
- Full docs & commands: **[GankList/README.md](GankList/)**

Maintainer note: after editing the addon, bump `## Version` in the toc, run `./build.sh`, and upload the resulting `GankList.zip` to a new [release](https://github.com/tuomopera/wow-addons/releases). The download link above always serves the latest release.
